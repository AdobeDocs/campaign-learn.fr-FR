# Prise en main pour les spécialistes du marketing

Ce guide présente les principales fonctionnalités de Campaign v8. Il est destiné aux marketeurs qui migrent de Campaign Standard vers Campaign v8.

Vous pouvez accéder à Adobe Campaign v8 à partir de la console cliente ou de l’interface utilisateur web. L’interface web vous permet de créer, gérer et exécuter des actions marketing clés.

>[!NOTE]
> Les versions de l’interface utilisateur Web d’Adobe Campaign fonctionnent sur un modèle de diffusion continu, ce qui permet une approche plus évolutive et progressive du déploiement des fonctionnalités. Consultez régulièrement les [notes de mise à jour](https://experienceleague.adobe.com/fr/docs/campaign-web/v8/release-notes/release-notes) pour connaître les dernières mises à jour.
